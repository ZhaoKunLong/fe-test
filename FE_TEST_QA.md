## 简答题
- 请简述Vue.js的生命周期钩子函数？
  ```markdown
   ### 请在此作答
   vue3的生命周期大概分为beforeCreated、created、beforeMounted、mounted、beforeUnmount、unmounted.
   - created(dom元素还未创建) 一般处理数据的获取和基本定义;
   - beforMounted(dom元素已经创建，但是还没有加载完成);
   - Mounted(dom元素已经加载完成) 可以进行相关的dom操作，比如控制元素位置改变等;
   - beforeUnmount(组件页面销毁之前) 常用于清空数据，定时器等，防止多个定时器执行;
   - unmounted(组件页面销毁)。
  ```
- 请简述Vue.js的组件化开发方式？
  ```markdown
  - 组件开发方式主要是抽离重复的业务代码, 封装成统一的组件进行调用;
  - 组件页面过长，拆分各部分业务界面进行封装，提高代码可读性和可维护性;
  ```
- 请简述CSS中的盒模型是什么，以及它的应用场景。
  ```markdown
  CSS 中的盒模型是指用于描述 HTML 元素的布局和尺寸的模型。它将每个元素视为一个矩形的盒子，由内容区域、内边距、边框和外边距组成。
  应用场景：
  - 布局
  - 边框样式
  - 响应式设计
  ```
- 请简述CSS中的选择器有哪些，以及它们的优先级。
  ```markdown
  - 元素选择器（Element Selector）：通过 HTML 元素名称选择元素，例如 p、div。
  - 类选择器（Class Selector）：通过类名选择元素，以点（.）开头，例如 .my-class。
  - ID 选择器（ID Selector）：通过元素的唯一 ID 选择元素，以井号（#）开头，例如 #my-id。
  - 属性选择器（Attribute Selector）：通过元素的属性选择元素，例如 [type="text"]。
  - 后代选择器（Descendant Selector）：通过元素的后代关系选择元素，例如 div p 选择 <p> 元素，其中 <p> 是 <div> 的后代元素。
  - 类型选择器（Type Selector）：通过元素的类型选择元素，例如 input[type="text"] 选择 type 属性为 "text" 的 <input> 元素。
  - 伪类选择器（Pseudo-class Selector）：通过元素的状态或特殊情况选择元素，例如 :hover、:first-child。
  - 伪元素选择器（Pseudo-element Selector）：通过元素的特殊部分选择元素，例如 ::before、::after。
  优先级： Id > 类 > 属性 > 伪类 > 元素
  ```
- 请简述CSS中的浮动和清除浮动是什么，以及它们的应用场景。
  ```markdown
  浮动是一种布局属性，用于将元素从正常的文档流中脱离，并使其根据指定的方向浮动到其容器的左侧或右侧。浮动元素会根据其相邻元素的位置进行调整，可以让其他元素环绕它。
  应用场景：创建多列布局、图片环绕文本、响应式布局等 

  清除浮动是一种技术，用于解决由浮动元素引起的布局问题。当容器中的元素浮动时，容器会丢失其高度，并可能导致后续内容覆盖浮动元素。清除浮动是一种方式，通过在容器内部添加额外的元素或应用特定的 CSS 属性，以恢复容器的高度和正确的布局。
  应用场景：防止容器布局塌陷。
  ```
- 请简述CSS中的定位和层叠顺序是什么，以及它们的应用场景。
  ```markdown
  css中的定位分为absolute、relative、fixed、sticky。
  - absolute: 绝对定位：脱离文本流，主要用于元素的位置排布，借助父元素的相对定位可以做出多种样式布局，没有的话就是基于docuemnt来定位;
  - relative: 相对定位：基于元素自身进行定位(不脱离文本流);
  - fixed: 固定定位：固定在某一位置，无论页面如何滚动，元素都不会改变位置;
  - sticky: 粘性定位：可以在滚动时定到某一位置也可以跟随滚动，一般用于header吸顶、广告悬浮和侧边框等;
  层叠顺序：z-index，主要用于页面的层级分布。类似于书页的一层一层排布，可以通过控制上层透明度大小等
  ```
- 请简述CSS中的响应式设计是什么，以及它的应用场景。
  ```
  有几种布局方法: grid布局、flex布局、百分比布局、rem布局
  - grid布局 适合均分的页面元素分布，可以通过media来控制页面元素的展示多少;
  - flex布局 flex主要用于元素对齐，和居中分布等;
  - 百分比布局 适用于可视化大屏，可以适配不同界面的分辨率界面;
  - rem布局 根据字体大小来进行页面的宽高控制
  ```

## 算法题
- 请实现一个二分查找算法。
   ```javascript
   //请在此作答
   function binarySearch(arr, target) {
    let low = 0;
    let high = arr.length - 1;

    while (low <= high) {
      let mid = Math.floor((low + high) / 2);
      if (arr[mid] === target) {
        return mid;
      } else if (arr[mid] < target) {
        low = mid + 1;
       } else {
        high = mid - 1;
      }
     }
      return -1; // 没有找到目标元素，返回 -1
    }
   ```
- 请实现一个链表的反转算法。
  ```javascript
   //请在此作答
  function reverseLinkedList(head) {
    let prev = null;
    let current = head;

    while (current !== null) {
      const next = current.next;
      current.next = prev;
      prev = current;
      current = next;
    }
    return prev;
  }
  ```
