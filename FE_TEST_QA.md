## 简答题

- 请简述 Vue.js 的生命周期钩子函数？
  ```markdown
  vue3: setup -> beforeCreate -> created -> beforeMount -> mounted -> beforeUnmount -> unmounted
  在 mounted -> beforeUnmount 此阶段的中间，还有 beforeUpdate -> updated
  ```
- 请简述 Vue.js 的组件化开发方式？

  ```markdown
  将一个整体大的应用或者说网页拆分成一个个组件，组件之间可以相互嵌套、也可以复用，相比以前的 Bootstrap+jQuery 时代，开发效率可以大大提高
  ```

- 请简述 CSS 中的盒模型是什么，以及它的应用场景。

  ```markdown
  html 中的每一个元素都可以看作一个盒子，盒子从外到里包括以下组成部分：

  1. margin
  2. border
  3. padding
  4. content 内容

  其中盒子的宽高度由 border、padding、内容三者决定，除此之外还取决于 box-sizing 属性的取值：

  1. content-box(default)，属性 width/height 只是内容的宽度/高度，实际长度等于 border+padding+content
  2. border-box，属性 width/height 等于 border+padding+content

  场景：布局或调整样式等
  ```

- 请简述 CSS 中的选择器有哪些，以及它们的优先级。

  ```markdown
  1. 元素选择器，比如 div、p、span 等标签名
  2. ID 选择器，比如 #content
  3. 类选择器，比如 .content
  4. 属性选择器，比如 a[href]、a[id^="id-"]
  5. 子孙后代选择器，比如 div p 表示 div 内的所有 p
  6. 直接后代选择器，比如 div > p 表示 div 内第一层级的 p
  7. 第一相邻兄弟选择器，比如 p + p 表示 p 相邻后面的第一个 p
  8. 相邻兄弟选择器，比如 p ~ p 表示 p 相邻后面的所有 p
  9. 通用选择器，比如 \* 表示所有的标签
  10. 伪元素选择器，比如 a:hover、p:nth-child
  11. 伪类选择器，比如 div::before、div::after
  12. 以上的任意组合

  优先级从高到低顺序如下：

  1. !important
  2. 行内样式(style)
  3. id 选择器
  4. 类选择器、属性选择器、伪类选择器
  5. 元素选择器、伪元素选择器
  6. 通用选择器
  ```

- 请简述 CSS 中的浮动和清除浮动是什么，以及它们的应用场景。

  ```markdown
  当一个元素被设置了 float 属性时(left/right)，会使其脱离文档流并向左或右浮动，直到到达父元素边框或另一个浮动的元素为止
  一般用于布局，但现在已经很少用，一般用 flex 或 grid 布局方式

  清除浮动是指 clear:both，设置该属性时，可以消除其他浮动元素对该元素的布局影响
  ```

- 请简述 CSS 中的定位和层叠顺序是什么，以及它们的应用场景。

  ```markdown
  position 属性包括以下取值：

  1. fixed，固定定位，根据视窗定位，其位置不会随着滚动条而变化
  2. absolute，绝对定位，根据其最近的父元素(非 static 定位，一般和上级 relative 进行配合)进行定位，会脱离文档流
  3. sticky，粘性定位，当元素位置在视窗内时，和 static 正常的一样，当超出视窗内，会表现得像 fixed 定位
  4. relative，相对定位，相对其 static 定位进行偏移
  5. static(默认)，正常文档流定位

  z-index 的生效要求 position 属性不能为 static，z-index 取值大的会遮挡住取值小的
  ```

- 请简述 CSS 中的响应式设计是什么，以及它的应用场景。

  ```
    响应式设计是指网站在不同的浏览设备(电脑/手机/平板)上，根据屏幕大小可以适配不同的样式，达到一致良好的浏览体验

    一般有以下方式：
    1. 媒体查询(@media)，根据屏幕大小阈值，设置不同的样式，使得在大屏幕上与小屏幕上浏览网页时可以正常显示内容
    2. 不使用绝对单位px，使用相对单位，常用的单位是rem、百分比、em、vh、vw
    3. 通过meta viewport设置width为固定值(px)，然后设置initial-scale为1
  ```

## 算法题

- 请实现一个二分查找算法。

  ```javascript
  //请在此作答
  const bsearch = (arr, target) => {
    let left = 0;
    let right = arr.length - 1;
    while (left <= right) {
      let mid = Math.floor((left + right) / 2);
      let midE = arr[mid];
      if (midE === target) {
        return mid;
      } else if (midE > target) {
        right = mid - 1;
      } else {
        left = mid + 1;
      }
    }

    return -1;
  };
  let arr = [1, 6, 5, 2, 8, 9, 3];
  // 二分查找要求数组从小到大排序好
  arr.sort((a, b) => a - b);
  // arr = [1, 2, 3, 5, 6, 8, 9]
  bsearch(arr, 6); // 4
  bsearch(arr, 8); // 5
  bsearch(arr, 2); // 1
  bsearch(arr, 100); // -1
  ```

- 请实现一个链表的反转算法。

  ```javascript
  //请在此作答
  // 所谓链表数据结构是指，一个元素中含有链接下一个(上一个)元素的地址信息，依次类推，就连成了一条链
  // 假设数据结构是这样的
  function Node(value, next) {
    this.value = value;
    this.next = next; // 存储下一个节点的引用
  }
  Node.prototype.toString = function () {
    let nextNode = this.next;
    let result = [this.value];
    while (nextNode) {
      result.push(nextNode.value);
      nextNode = nextNode.next;
    }
    return result.join(" -> ");
  };
  Node.prototype.reverseChain = function () {
    let prev;
    let node = this;
    while (node) {
      let next = node.next;
      node.next = prev;
      prev = node;
      node = next;
    }
  };

  // 构造测试链表
  let nodes = [1, 2, 3, 5].map((i) => {
    return new Node(i);
  });
  nodes.forEach((i, idx, arr) => {
    i.next = arr[idx + 1];
  });
  let first = nodes[0];
  console.log(first.toString()); // 1 -> 2 -> 3 -> 5
  first.reverseChain();
  let last = nodes[nodes.length - 1];
  console.log(last.toString()); // 5 -> 3 -> 2 -> 1
  ```
