## 简答题
- 请简述Vue.js的生命周期钩子函数？
  ```markdown
   ### Vue 2
   1.beforeCreate: Vue实例创建前，data数据和methods方法都还未初始化。
   2.created：Vue实例创建后，数据已经被初始化。
   3.beforeMount：Dom渲染之前。
   4.mounted：Dom渲染之后。
   5.beforeUpdate： 数据更新前。
   6.updated： 数据更新后。
   7.beforeDestroy： 组件销毁前（常在此时清楚定时器）。
   8.destroyed： 组件销毁后。
   当使用keep-alive缓存后会增加activated和deactivated这两个生命周期函数。
   activated：组件激活时被调用。
   deactivated： 组件停用时调用。
   ### Vue 3
   1.setup: 初始化之前执行。
   2.onBeforeMount： 相当于Vue2的beforeMount，Dom渲染前执行。
   3.onMounted：相当于Vue2的mounted，Dom渲染之后执行。
   4.onBeforeUpdate：相当于Vue2的beforeUpdate，组件更新前执行。
   5.onUpdated：相当于Vue2的updated，组件更新后执行。
   6.onBeforeUnmount：相当于Vue2的beforeDestroy。
   7.onUnmounted：相当于Vue2的destroyed。
   当使用keep-alive缓存后会增加onActivated和onDeactivated这两个生命周期函数，同Vue2的activated和deactivated。
  ```

- 请简述CSS中的盒模型是什么，以及它的应用场景。
  ```markdown
  1.标准盒模型（content-box）:盒子尺寸是由元素的宽高+border+padding。这是css默认的盒模型，可以更方便计算盒子的尺寸。
  2.怪异盒模型（border-box）:盒子尺寸是由元素的宽高决定。适合在特定条件下盒子大小包含padding和border的情况，很多UI设计的就需要使用怪异盒模型。
  
  ```

- 请简述CSS中的浮动和清除浮动是什么，以及它们的应用场景。
  ```markdown
  浮动：让元素脱离文档标准流，实现横向布局。最多的应用场景就是把纵向布局改为横向布局。
  清除浮动：解决浮动带来的一些问题，比如高度塌陷，内容重叠。可使用clear:both清除，也可以给父元素设置一个高度，添加伪类等。

  ```
- 请简述CSS中的定位和层叠顺序是什么，以及它们的应用场景。
  ```markdown
  ### 定位
  1.相对定位：position: relative。相对原来的位置进行调整，不会影响其它元素。应用场景:可提高层级，调整元素的位置。
  2.绝对定位：position: absolute。脱离标准文档流，提高层级，相对于最近定位的父元素进行定位。应用场景:可实现水平垂直布局，遮罩层，标签；提高层级；定位到盒子想要的位置；
  3.固定定位：position: fixed。相对于浏览器窗口进行定位。应用场景：实现返回顶部按钮，底部顶部导航，模态框等
  ### 层叠顺序
  层叠顺序是指当多个元素发生重叠时，决定哪一个显示在上面，哪一个显示下面的规则。从低到高依次是：
  背景和边框，负的z-index,块级盒子，浮动盒子，行内盒子，z-index=0,z-index为正的。

  ```
- 请简述CSS中的响应式设计是什么，以及它的应用场景。
  ```markdown
  响应式设计是指在不同分辨率的屏幕下，网页的布局自适应调整，包括h5和web。
  使用场景：
  1.对一些大屏项目针对在不同分辨率的屏幕尽可能完美展示数据，尽量不出现滚动条，特别是横向滚动条。
  2.对后h5和web进行自使用调整。
  使用方式：
  1.通过媒体查询media，这种方式可以设置屏幕的分辨率临界值，然后通过样式进行调整布局。
  2.通过监听屏幕大小以及和设计图的尺寸对比进行等比例的放大缩小。
  3.使用rem,vw以及百分比单位设置大小。

  ```

        if (a > b) return 1
        else return -1
    })

    function find(arr, num) {
        if (arr.length === 0) {
            return -1
        }
        if (arr.length === 1) {
            if (arr[0] === num) {
                return 0
            } else {
                return -1
            }
        }
        const index = Math.floor(arr.length / 2);
        if (arr[index] < num) {
            const result = find(arr.slice(index), num)
            if (result == -1) {
                return -1;
            }
            return result + index
        } else if (arr[index] == num) {
            return index
        } else {
            return find(arr.slice(0, index),num)
        }

    }
